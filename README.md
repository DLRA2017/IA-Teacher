# Talking English - Tutor de Inglês com IA (Nível A2)

## 📝 Sobre o Projeto
O **Talking English** é uma aplicação interativa desenvolvida para estudantes de inglês no nível A2 (Pré-intermediário). Utilizando a API do **Google Gemini**, o sistema atua como um tutor paciente (Mr. Deivis) que adapta o vocabulário e a complexidade gramatical para garantir que o aluno consiga praticar sem frustrações.

O diferencial deste projeto é a integração da **Web Speech API**, que permite ao aluno treinar o *Listening* ao ouvir as respostas da IA em tempo real.

## 🚀 Tecnologias e Recursos
* **Inteligência Artificial:** Integração com Google Gemini 1.5 Flash via REST API.
* **Text-to-Speech (TTS):** Síntese de voz nativa do navegador para prática auditiva.
* **Frontend Moderno:** Desenvolvido com **Tailwind CSS** para um design responsivo e intuitivo.
* **Engenharia de Prompt:** Instruções de sistema (System Instructions) para garantir o comportamento pedagógico da IA.

## ⚙️ Como Utilizar
Por questões de segurança e boas práticas de desenvolvimento, as credenciais de API não estão incluídas neste repositório.

1. Obtenha sua chave gratuita em [Google AI Studio](https://aistudio.google.com/).
2. Abra o arquivo `index.html`.
3. Localize a variável `const apiKey = ""` e insira sua chave entre as aspas.
4. Salve e abra o arquivo em seu navegador.

## 🔒 Segurança de Dados
Este projeto foi desenvolvido com foco em **Cloud Security**. Seguindo os aprendizados do curso de Sistemas de Informação, o código-fonte foi higienizado para evitar o vazamento de credenciais (Hardcoded API Keys). Recomenda-se que o usuário utilize restrições de IP ou referenciador HTTP ao ativar sua chave no Google Cloud Console.
